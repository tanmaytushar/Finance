# Finance

## Project Details 
- Electron.js to compile for the desktops
- SQLlite3 as database interfacing with sql.js module
- React with CDN for the UI and Bulma CSS for the components and style.

### Local Setup
-   Clone the repo
-   `npm install`
-   `npm run start`
